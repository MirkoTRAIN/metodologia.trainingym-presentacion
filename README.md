# Las 4 palancas del negocio fitness · Trainingym

Web pública con la metodología Trainingym de las 4 palancas para hacer crecer un negocio fitness:

1. **Captación**
2. **Ahorro de tiempo**
3. **Fidelización**
4. **Ingresos**

> 🌐 Web standalone. Diseñada para presentar la metodología a potenciales clientes.

---

## 🎯 Cómo funciona

- Las **4 palancas** se muestran en formato lineal (una al lado de la otra)
- Cada palanca tiene su **título arriba** (Palanca 01 / 02 / 03 / 04)
- Al hacer **click en una palanca**, esa hace zoom y las otras se quedan compactas a los lados
- Botones arriba para alternar entre palancas rápidamente
- Cada palanca tiene **sub-temas clicables** que abren un modal con detalle completo

### Ejemplo de uso comercial

1. Comparte el link con un potencial cliente: *"mira esto antes de la reunión"*
2. El cliente entra y ve **las 4 palancas de un vistazo**
3. Hace click en la que más le interese (o todas) y ve el detalle
4. Llega al CTA al final → te contacta para hablar

## 🚀 Despliegue en GitHub Pages

1. Crea un repo público nuevo: [github.com/new](https://github.com/new) → `palancas-trainingym`
2. Sube `index.html`, `README.md`, `.gitignore`
3. **Settings → Pages**: Source `Deploy from a branch` → Branch `main` `/` (root) → **Save**
4. Espera 1–2 min → URL pública lista
5. Comparte la URL con potenciales clientes
